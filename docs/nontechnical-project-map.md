# Non-Technical Project Map

This map separates the project into three layers so it is easier to discuss and evolve.

## Layer 1: Cultural Method

This is the intellectual spine.

The project asks:

- How do archives decide what we can find?
- What does a keyword like "cloud" reveal or hide?
- How do institutional APIs shape cultural visibility?
- What happens when human intuition interrupts automated classification?
- Can image proximity create new cultural meaning?

This layer should remain visible in the future app. The tool should not become only a database.

## Layer 2: Technical Pipeline

This is the machinery.

The pipeline is:

1. Search institutional archives.
2. Download images and metadata.
3. Add subjective images manually.
4. Combine the collection.
5. Create visual clusters.
6. Let a curator interpret, rename, split, merge, and sequence clusters.
7. Export or publish the result.

In the current repo, steps 1 and 2 exist as partial scripts. Steps 3 through 7 exist mostly as methodology documents.

## Layer 3: Future Living Atlas

This is the custom tool we eventually want.

The atlas should let you:

- Import your own image collection.
- Import from Google Drive.
- Import from URLs or CSV.
- See images arranged by visual and semantic affinity.
- Add subjective groupings that can disagree with the machine.
- Write curatorial notes.
- Create narrative paths.
- Publish selected constellations publicly while keeping the working archive private.

This layer should be ambitious, but it should grow from understanding the original first.

## Simple Vocabulary

- Archive: the whole body of images and metadata.
- Metadata: the words and fields attached to an image.
- API: a structured doorway into an institution's database.
- Scraper: a small program that collects data from an API or website.
- Cluster: a group of images that the machine sees as related.
- Recuration: the human act of accepting, rejecting, renaming, reshaping, or sequencing machine groups.
- Atlas: a spatial image system for thinking, not only storing.
